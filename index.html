⚠ 若發現「重整網頁後資料就不見」，請確認以下設定：

  1. 開啟匿名認證： 進入 Firebase Console ➡ Authentication ➡ Sign-in method ➡ 啟用 「匿名 (Anonymous)」 登入方式。
  2. 設定安全規則： 進入 Firestore Database ➡ 規則 (Rules) ➡ 將內容修改並發布為：
    allow read, write: if request.auth != null;
  3. 新增 GitHub 授權網域： 到 Authentication ➡ 設定 (Settings) ➡ 授權網域 ➡ 新增 您的帳號.github.io（不可帶有 https://）。
